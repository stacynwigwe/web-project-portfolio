# Website Portfolio – CMS & Web Implementation Projects

This repository showcases three website projects I designed and implemented using **WordPress** and **no-code/low-code CMS platforms**. Each project demonstrates my ability to translate **business and educational needs** into functional, user-centered web systems, including **CMS configuration, content architecture, and workflow design**.

---

## 1️Tutoring Website – WordPress CMS Implementation

### Project Overview

This project represents an end-to-end WordPress website build for a tutoring organization serving grades 3–8 across Broward County. I owned the design, build, and backend configuration, delivering both public-facing content and authenticated user functionality.

### What I Did

- Designed the website layout, branding, and overall content structure  
- Implemented WordPress CMS functionality from the ground up  
- Engineered student registration and login workflows  
- Configured dashboard-based user access and authentication
    
- Built and structured service pages to support enrollment and program discovery  
- Translated instructional and operational requirements into usable web features  

### Tools & Platform

- WordPress  
- Custom themes and plugins  
- Forms, user roles, and authentication  
- CMS content modeling
